# Facial Recognition 

The general breakdown of the methodology of the repositort will be explained in FacialRecognition.docs

Using tensorflow keras and sklearn this repository explores the use of different classifier to classifiers whether two faces 
are the same or not.

The classifiers used in this repository are: SVM, gradient boosted trees,ExtraTreeClassifier, DecisionTreesClassifier
,Randomforest, DNN, Bayesian, BaggingClassifier, Votingclassifer
